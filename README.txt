# README
This is a repository for storing analysis code and other materials for OriSeg.

## Set up
*Requires anaconda installation

	1) Clone repository
	2) Set up python environment: python env_config.py
	3) Activate environment: conda activate oriseg

## Directories
	code/
		analysis code
	
	papers/
		manuscripts associated with OriSeg

	presentations/
		presentations on OriSeg
